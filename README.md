## 🚀 Technical Features

* **Comprehensive Form Validation:** Custom JavaScript logic to validate text inputs, email formats, radio buttons, and checkboxes before submission.
* **Success State Feedback:** A dedicated success message that appears upon valid form submission, improving user engagement.
* **Mobile-First Workflow:** A fluid layout that transitions from a single-column mobile stack to a multi-column desktop grid.
* **Accessible UI Patterns:** Built with semantic HTML5 elements and clear focus states to ensure the form is navigable for all users.
* **Modern CSS Styling:** Use of CSS Custom Properties for theme management and Flexbox/Grid for precise element alignment.

## 🛠️ Tech Stack

- **Structure:** HTML5 (Semantic & Accessible)
- **Styling:** CSS3 (Flexbox, Grid, Custom Properties)
- **Logic:** Vanilla JavaScript (ES6+)
- **Deployment:** Vercel

## 🧠 Key Learnings

Building this contact form provided deep insights into **form state management** and **client-side error handling**.

**Main Takeaways:**
- Mastering **DOM selection and event listeners** to provide real-time validation feedback.
- Implementing custom styling for radio buttons and checkboxes while maintaining accessibility.
- Handling complex layouts where input fields change positioning based on viewport width.
- Developing a clean, modular JavaScript structure to manage multiple input types efficiently.

## 💻 Getting Started

### Prerequisites
A modern web browser and a code editor.

### Local Execution
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/itsgsantos/contact-form-main.git](https://github.com/itsgsantos/contact-form-main.git)
    ```
2.  **Navigate to the folder:**
    ```bash
    cd contact-form-main
    ```
3.  **Run the project:**
    Open `index.html` in your browser.
